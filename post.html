Spending Management System

This was a project I started back in June 2020 when I first learned Ruby. This was my first ever solo project and I am proud of how it turned out.

The system is utilising Postgres on the backend that allows it to save the users data and also allows for timestamp generation, this makes it even easier for the user to track spending.

Sinatra a Ruby web framework, is being used to handle request routing and provide a mechanism for the application to respond to different URLs with the right response within the program.

On the front end a mixture of Ruby, HTML & CSS was used. I tried to design the front end in such a way that it would be recognizable to the user, so the systems UI is heavily inspired by modern online banking systems.

The system allows users to track their daily expenditure, while keeping tabs on varying merchants and shopping categories.

Technologies:
  • - Ruby
  • - HTML/CSS
  • - PostgreSQL
  • - Sinatra
